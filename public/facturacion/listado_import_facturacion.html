Procedimiento de Facturación

Proceso de Gestión de Facturación

1. Registro de Importación

• Se registra la importación con todos los datos básicos

• Se asigna un número de despacho único

• Se especifica el tipo de carga y productos

2. Estados de Facturación

• REGISTRADO: Despacho facturado correctamente

• NO REGISTRADO: Pendiente de facturación

3. Gestión de Documentos

• Los PDFs de solución están disponibles para consulta

• Se pueden verificar los documentos de facturación

• Registrar observaciones sobre el estado de facturación

4. Actualización de Estados

• Utilizar el botón "Editar" para modificar información

• Actualizar estado de registro de facturación

• Agregar observaciones relevantes

Notas Importantes:

• Mantener actualizada la información de facturación

• Los documentos PDF deben estar siempre disponibles

• Comunicar cualquier cambio de estado inmediatamente

• El sistema permite filtrar por fechas y número de despacho
