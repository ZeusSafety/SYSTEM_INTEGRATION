📖 Procedimiento de Uso

Este sistema permite gestionar y dar seguimiento a las solicitudes de manera rápida y organizada. A continuación, se detalla su funcionamiento:

🔎 Filtros
  • Filtra las solicitudes por Área, Colaborador, Estado y Incidencia.
  • El filtro de Colaborador funciona en tiempo real mientras escribes.
  • Se permite hacer Filtros Combinados funciona en tiempo real.
✍️ Responder Solicitudes
  • En la columna Acciones podrás abrir el formulario de respuesta.
  • Completa los campos de Respondido por, Respuesta e Informe (opcional).
  • Puedes adjuntar un archivo PDF y luego visualizarlo con el botón Ver archivo.
🚨 IMPORTANTE: SOLO TIENEN MÁXIMO 48 HORAS PARA RESPONDER O ATENDER UNA SOLICITUD/INCIDENCIA.
🔄 Reprogramaciones
  • Si es necesario reprogramar, activa el checkbox de Reprogramación.
  • Se permite máximo 1 reprogramación.
  • Si se trata de un caso complicado, se permite un máximo de 3 reprogramaciones.
  • Para agregar una nueva reprogramación, usa el botón verde ➕ en la primera o segunda reprogramación.
  • En los campos de reprogramación se puede escribir el motivo y opcionalmente un link a documentos.
📑 Visualización
  • Si una solicitud tiene reprogramaciones, en la columna Con reprogramación se mostrará SI.
  • Al lado aparecerá un botón para abrir el detalle de todas las reprogramaciones registradas.
